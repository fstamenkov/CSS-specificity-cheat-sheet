Table of Selectors

Example Tag Selector type Description
#example ID Includes only ID selectors. Declared with type="example".
.myClass Class Includes only class selectors. Declared with class="myClass".
p, h1, td Type Target an HTML element such as an <h1>
* Universal Selector Affects styling of all elements.

Arranged by decreasing specificity.

Pseudo-classes and pseudo-elements

These classes and elements style certain states of an element. The :hover pseudo-class selects an element when it is being hovered by a mouse pointer:

a:hover

A pseudo-element which selects a certain part of an element rather than the element itself.

p::first-line
